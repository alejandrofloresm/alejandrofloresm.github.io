Índice de prácticas

A continuación encontrarás un listado de prácticas:

  • Física (3D): aprenderás los elementos básicos de física en 3D dentro de Unity3D
  • Física (2D): aprenderás los elementos básicos de física en 2D dentro de Unity3d.
  • Animaciones (2D): conocerás los elementos básicos para manejar las animaciones de personajes en 2D, aquí realizarás las mecánicas básicas de un juego tipo "Platformer".
  • Consola Git (Github): Conocerás como manejar una consola de administración de git usando Github como servidor de origen.
  • Consola Git (Bitbucket): Conocerás como manejar una consola de administración de git usando Bitbucket como servidor de origen.
